===================================
Test cases for MyList class
===================================

Import the class:
    >>> MyList = __import__('1-my_list').MyList

Test basic functionality:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test with negative numbers (unsorted):
    >>> my_list2 = MyList()
    >>> my_list2.append(-1)
    >>> my_list2.append(4)
    >>> my_list2.append(-10)
    >>> my_list2.append(3)
    >>> my_list2.append(-5)
    >>> my_list2.print_sorted()
    [-10, -5, -1, 3, 4]
    >>> print(my_list2)
    [-1, 4, -10, 3, -5]

Test with empty list:
    >>> my_list3 = MyList()
    >>> my_list3.print_sorted()
    []

Test with one element:
    >>> my_list4 = MyList()
    >>> my_list4.append(1)
    >>> my_list4.print_sorted()
    [1]

Test with already sorted list:
    >>> my_list5 = MyList()
    >>> my_list5.append(1)
    >>> my_list5.append(2)
    >>> my_list5.append(3)
    >>> my_list5.print_sorted()
    [1, 2, 3]

Test with reverse sorted list:
    >>> my_list6 = MyList()
    >>> my_list6.append(5)
    >>> my_list6.append(4)
    >>> my_list6.append(3)
    >>> my_list6.append(2)
    >>> my_list6.append(1)
    >>> my_list6.print_sorted()
    [1, 2, 3, 4, 5]

Test with duplicate values:
    >>> my_list7 = MyList()
    >>> my_list7.append(3)
    >>> my_list7.append(1)
    >>> my_list7.append(3)
    >>> my_list7.append(1)
    >>> my_list7.print_sorted()
    [1, 1, 3, 3]

Test that MyList inherits from list:
    >>> my_list8 = MyList()
    >>> isinstance(my_list8, list)
    True

Test with initialization:
    >>> my_list9 = MyList([3, 1, 2])
    >>> my_list9.print_sorted()
    [1, 2, 3]

Test with only negative numbers:
    >>> my_list10 = MyList()
    >>> my_list10.append(-5)
    >>> my_list10.append(-1)
    >>> my_list10.append(-3)
    >>> my_list10.print_sorted()

Test with zero:
    >>> my_list11 = MyList()
    >>> my_list11.append(0)
    >>> my_list11.append(-1)
    >>> my_list11.append(1)
    >>> my_list11.print_sorted()

Test that print_sorted method exists:
    >>> my_list12 = MyList()
    >>> hasattr(my_list12, 'print_sorted')
